Built-in Tags

The registry (not to be confused with the Windows registry) consists of what the official modding documentation refers to as "nodes", which are key and value pairs where both are strings.

Registry nodes are not to be confused with commands, which follow the same naming conventions but are handled entirely separately.

Tags

"Non-engine" means it is only ever get or set by Lua scripts, and never by the engine.

fxbreak=[liquid or gas, l/g][intensity, 1-9](color)
Shape tag. Generate particle effects when shape is broken. Valid colors for liquids are yellow, black, blue, green, and brown. Valid colors for gasses are yellow, blue, and green. Default Color is white. This is handled in fx.lua. Non-engine.
Added in 0.9.6 (8498817).
fxsmoke=p[size, 1-9][color]
Location tag. When positioned near a shape, makes it spawn in smoke particles. Valid colors are brown and subtle. This is handled in fx.lua. Non-engine.
Added in 0.9.6 (8498817).
fxflicker=[time, 1-9]
Light tag. Make light flicker. This is handled in fx.lua. Non-engine.
Added in 0.9.6 (8498817).
explosive=[size, 0.5-4.0]
Body/Shape tag. Entity explodes when broken. If applied to an already broken shape, it instantly blows up. This is handled by the engine.
unbreakable
Body/Shape tag. Entity unbreakable, but still markable by spray paint or burn marks.
nocull
Shape tag. Shape is never culled.
interact=(interact text)
Body/Shape tag. Makes it so when the player is close to the tagged entity, a prompt shows up to interact with it. GetPlayerInteractShape/Body only works on entities with this tag.
autobreak
Body tag. Makes body require a smaller inertia to trigger a break event. This is used/set/handled by the engine for vehicle body(ies?).
Added in 0.9.0 (7801522).
breakall
Shape tag. Automaticly break the other shape that touches the this tagged shape. This is used/set in robot.lua and used/set by the engine if a explosive entity is thrown? handled by the engine?
Added in 0.9.0 (7801522).
inherittags
Body/Shape tag? takes all the tags from tagged entity and set them on its debris. It will recursively do this.
awake
Body/Shape tag. When the level loads, the tagged body/shape body will not be asleep. Non-engine.
Added in 0.7.0 (6551167).
map
Body tag. Marks body on map. Defaults to a blue color. Non-engine.
Added in 0.4.5 (5748542).
invisible
Shape tag. Make tagged shape invisible, however still impacts the shadow volume.
Added in 1.3.0 (10148769).
nonav
Shape tag. Shape is entirely disreguarded by pathfinding.
Added in 0.9.0 (7801522).
night
Light tag. Light will be turned off if the nightlight environment property is currently false.
nosnow
Shape tag. Shape does not have any snow on it when the level loads.
Added in 0.7.1 (6627985).
nodrive
Vehicle tag. Vehicle is not drivable to the player
reversebeep
Vehicle tag. Vehicle makes beep sound when reversing
Added in 0.4.5 (5748542).
boat
Vehicle tag. Makes vehicle a boat, with additional boat physics applied ane able to use propeller locations, but lose ability to use wheel?
Added in 0.5.1 (5987914).
dumptruck
Vehicle tag. All hinge joints on vehicle are controllable with vehicle_raise/vehicle_lower, display bed controls
frontloader
Vehicle tag. All hinge joints on vehicle are controllable with vehicle_raise/vehicle_lower, display arm controls
skylift
Vehicle tag. All hinge joints on vehicle are controllable with vehicle_raise/vehicle_lower, display lift controls
skylift
Vehicle tag. All hinge joints on vehicle are controllable with vehicle_raise/vehicle_lower, display crane controls. Hook is also controllable.
forklift
Vehicle tag. All prismatic joints on vehicle are controllable with vehicle_raise/vehicle_lower, display forklift controls.
forklift
Vehicle tag. Display tank controls. Non-engine.
Added in 0.9.0 (7801522).
hook
Body?/Shape tag. Shape body acts as vehicle hook which can be controlled by vehicle_action.
player
Location tag. Defines where the entry point and first person camera are for a vehicle.
camera
Location tag. Defines where the 3rd person camera is. Defaults to player location. This tag have had its behaviour changed after Version? and was not usable during Version?
exit
Location tag. Defines where the player will exit vehicle to. Defaults to left of player location.
exhaust=(amount)
Location tag. Defines where/what direction exhaust is emitted from the vehicle.
vital
Location tag. When voxels at this position are broken, the vehicle is instantly broken(?). Defines where engine smoke comes from.
propeller
Location tag. Defines where the propeller is in a boat.

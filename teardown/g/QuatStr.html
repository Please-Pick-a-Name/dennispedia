QuatStr

Returns a human readable string with the values of a quaternion. Approximately equivalent to string.format("{%s, %s, %s, %s}", quat[1], quat[2], quat[3], quat[4]). Like VecStr, but for quaternions. See that article for details.

This function is not officially documented.

Arguments

  1. Quat (table): Quaternion to turn into a string

Return values

  1. string: Approximate string representation

ParticleOrientation

Set orientation of particles to be spawned. Sole argument can contain normalup, up, or fixed (mutually exclusive, prioritized in that order) and optionally flat.

normalup
Face the camera, with a random roll each frame.
up
Face up.
fixed
Face the camera, but with a constant roll.
flat
The particle will be flat instead of a pyramid shape.

This function was added in 0.9.6 (8498817).

This function is not officially documented.

Arguments

  1. string: Flags

Return values

None.

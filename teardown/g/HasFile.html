HasFile

Returns whether a file exists at the specified path. Respects MOD/, LEVEL/, and RAW:.

This function is not officially documented.

This function was added sometime between 0.5.2 (6001321) and 0.9.2 (patch 1) (7913435).

Arguments

  1. string: Path relative to current file, or starting with MOD/, LEVEL/, or RAW:

Return values

  1. boolean: Whether a file or directory exists at the specified location

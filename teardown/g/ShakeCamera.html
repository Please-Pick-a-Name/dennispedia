ShakeCamera

Shakes the camera for the specified amount of time. The longer the duration, the more intense the shaking.

This function was added in 0.9.0 (7801522).

This function is not officially documented.

Arguments

  1. number: Duration in seconds. Non-number values are treated as 0. Positive infinity will break the game and requires relaunching to fix.

Return values

None.

Versions

The version code is what the Google Play client must present to the server to download an app. In this way, it is similar to a Steam manifest ID or buildid. I am unsure if there is an equivalent to it on Apple's App Store.
You can use it to install that particular version directly from Google Play by using Aurora Store, like so.
